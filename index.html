This webpage provide a simple step by step tutorial on how to perform basic polygenic risk score analyses. By following this tutorial, you should be able to perform basic quality control of the summary statistics and target genotype files, and will be able to carry out simple polygenic risk score analyses using only bash, R and plink.

Note

This tutorial is based on linux operation system. Users who would like to perform polygenic risk score analyses on their Mac or Windows machine will need to change some of the commands.

We might include tutorial for the other operation systems in the near future.

Requirements¶

To follow the tutorial, you will need the following programs installed:

  1. R (version 3.2.3+)
  2. PLINK 1.9
